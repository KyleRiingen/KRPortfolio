More Projects

Uno AI: C++

For this project I created a CPU controlled bot which plays the card game UNO. If you are unfamiliar with this card game each player is given cards with distinct colors and numbers. You can play cards, or remove them from your hand when there is either a matching color or number in play. The player who removes all cards from their hand first wins. My program is designed to aim for the lowest score possible, given the games reliance on luck.

Socket Program: Java

This program consists of a simple socket connection. Once the client creates a connection with the server, the user is asked to insert a large integer. Once an integer is given, the server performs a series of simple mathematic operations. This is repeated until the integer is one digit in length

Binary Search Tree: C++

This program is used to illustrate a binary search tree. The program consists of node classes which are then organised in a manner to form a BST. If unfamiliar, a binary search tree consists of nodes, which each have two "child nodes." The child nodes are either left nodes or right. The child node on the left will always be greater than the connect node, aka root node, and the node on the right will always be greater
